# Tribute to Ki Hajar Dewantara

**Nama:** Ahmad Ridwan Hasibuan
**Proyek:** Tugas Hardiknas HIMATIF UISU

## Deskripsi
Website ini merupakan landing page dedikasi untuk mengenang jasa Ki Hajar Dewantara dalam dunia pendidikan Indonesia.

## Mengapa Memilih Tokoh Ini?
Saya memilih Ki Hajar Dewantara karena filosofi pendidikan beliau, yaitu "Ing Ngarsa Sung Tulada, Ing Madya Mangun Karsa, Tut Wuri Handayani," masih sangat relevan dengan semangat belajar mahasiswa Teknik Informatika saat ini. Beliau mengajarkan bahwa pendidikan adalah kunci kemerdekaan bangsa.
